# 📚 Library Management System (Java)

A simple **Library Management System** built using **Java** that allows users to manage books through a console-based interface.

This project demonstrates core concepts of **Object-Oriented Programming (OOP)** and **Java Collections**.

---

## 🚀 Features

* ➕ Add new books
* 📖 View all books
* 📕 Issue a book
* 📗 Return a book
* 🔄 Real-time status (Available / Issued)
* 🧠 Menu-driven console interface

---

## 🛠️ Tech Stack

* **Java**
* **OOP Concepts** (Classes, Objects)
* **ArrayList (Collections Framework)**

---

## ⚙️ How It Works

* Each book is represented as an object with:

  * Title
  * Author
  * Issue status

* Books are stored in an `ArrayList`

* User interacts using a menu-driven system

* Operations are performed using loops and condition checks

* Case-sensitive book search
* Single-user system
